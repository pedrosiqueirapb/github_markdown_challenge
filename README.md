# Desafio Formação Github Certification: Colaboração e Markdown

`PLATAFORMA` DIO

`FORMAÇÃO` Github Certification

`MÓDULO` Introdução aos Conceitos de Git e Github

Repositório criado para o desafio de projeto **Explorando Colaboração e Markdown** da Formação Github Certification da DIO. O desafio inclui a criação de um repositório, adição de um ou mais colaboradores, clonagem na máquina local, utilização da linguagem markdown, criação de *issues* e *pull requests*.
